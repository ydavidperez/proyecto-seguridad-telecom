# Auditoría ISO 27001

## Evaluación de Controles

### Inventario de Activos

¿El modelo YOLO logró listar routers y switches?

Sí. El sistema identifica automáticamente dispositivos de red mediante visión artificial.

### Uso Aceptable de Activos

El software fue desarrollado únicamente para inventario y auditoría de telecomunicaciones.

### Transferencia de Información

Los datos son protegidos mediante cifrado híbrido RSA + AES.

### Etiquetado de Activos

Las imágenes detectadas sirven como evidencia visual para auditoría.

### Protección contra Malware

Se recomienda ejecutar el proyecto sobre equipos protegidos con antivirus y firewall.

| Dominio | Implementación |
