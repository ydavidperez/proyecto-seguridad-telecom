# Sistema de Inventario Visual y Auditoría de Seguridad para Telecomunicaciones

Proyecto académico orientado a detección YOLOv8, criptografía y auditoría ISO 27001.

## Módulos

### YOLOv8
- Router
- Switch
- Firewall
- Servidor
- Access Point

### Criptografía
- SHA-256
- AES-256
- RSA

### Auditoría
- Inventario de activos
- Transferencia segura
- Etiquetado visual
- Protección contra malware

## Matriz de riesgos

| Riesgo | Mitigación |
|---|---|
| Interceptación | RSA + AES |
| Pérdida de integridad | SHA-256 |
| Malware | Antivirus |

## ISO 27001

| Control | Estado |
|---|---|
| Inventario | Cumple |
| Criptografía | Cumple |
| Uso aceptable | Cumple |
