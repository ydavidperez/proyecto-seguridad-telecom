# Proyecto Final - Seguridad de la Información

## Sistema de Inventario Visual y Auditoría de Seguridad para Telecomunicaciones

### Objetivo General
Desarrollar un sistema automatizado que detecte activos de telecomunicaciones mediante YOLO, proteja la información mediante criptografía y evalúe el cumplimiento de la norma ISO 27001:2022.

## Módulo 1 - Detección de Activos con YOLO
Clases: Router, Switch, Firewall, Servidor y Hub.

Dataset propuesto: 100 imágenes distribuidas en cinco clases.

Herramientas utilizadas:
- Roboflow
- Google Colab
- GitHub
- Teléfono móvil para captura de evidencias

## Módulo 2 - Criptografía
- AES para confidencialidad.
- RSA para intercambio seguro de claves.
- SHA-256 para integridad.

## Módulo 3 - Auditoría ISO 27001:2022
Evaluación de controles:
- Inventario de activos.
- Uso aceptable de activos.
- Transferencia de información.
- Etiquetado de activos.
- Protección contra malware.

## Matriz de Riesgos
Incluye riesgos asociados a pérdida, interceptación o modificación de evidencias.

## Conclusión
La solución integra Inteligencia Artificial, Criptografía y Auditoría de Seguridad para fortalecer la gestión de activos de telecomunicaciones.
